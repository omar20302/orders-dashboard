# MAD Orders Dashboard V5

Streamlit dashboard for analyzing orders by branch, hour, and fillings.

## Deploy on Streamlit Community Cloud

1. Create a GitHub repository.
2. Upload these files:
   - `app.py`
   - `requirements.txt`
3. Go to Streamlit Community Cloud.
4. Create a new app from the GitHub repo.
5. Main file path: `app.py`
6. Deploy.

## Google Sheet setup

For the simple public-link method:
- Share the Google Sheet as: `Anyone with the link can view`
- Open the tab that contains the orders
- Copy the Google Sheet URL
- Paste it in the app

The app can filter by pickup/delivery date using the selected date column.
